Netlify × GitHub 連携手順

GitHubのリポジトリからNetlifyに自動デプロイする最短ガイド。

← トップに戻る

① リポジトリを準備 GitHub

  • 公開したいソースを main ブランチに置く（静的なら index.html だけでOK）。
  • 必要なら /docs や /public にPDFなどのアセットを配置。

② Netlify に接続 Netlify

  • Netlifyにログイン → Add new site → Import an existing project。
  • Git provider で GitHub を選択し、リポジトリを指定。
  • ビルド設定（静的サイトなら空でOK / または Publish directory に . や dist）。

③ デプロイと確認

  • 初回デプロイ完了後、発行されたURLにアクセスして表示を確認。
  • 以降は git push で自動デプロイ。
# 例
git add .
git commit -m "Initial deploy"
git push origin main

補足：PDF掲載のコツ

PDFはリポジトリ直下に置けば /your.pdf でアクセス可能。ページ内に埋め込む場合は:

<iframe src="/seo-training.pdf" width="100%" height="800" frameborder="0"></iframe>
